Why Tau?

🧠 Familiar Syntax

Tau uses a syntax that feels natural to seasoned programmers. This makes the language approachable, minimizing the learning curve and allowing you to focus on what makes Tau unique.

🔗 Interoperability

Seamlessly interface with C libraries and code. Tau's ABI compatibility with C means you can reuse existing libraries without wrappers or bindings, making integration smooth and efficient.

⚙ Low-Level Control

Tau gives you precise control over memory and system resources. You can manage allocation manually, work with raw pointers, and write performant code with no overhead.

✨ High-Level Constructs

While close to the metal, Tau also supports high-level abstractions such as generics, interfaces and generators, enabling expressive code without sacrificing performance.

🧬 Metaprogramming

Tau includes compile-time metaprogramming tools that let you write code that writes code. Generate functions, types, and logic based on static input to reduce repetition and increase flexibility.

📦 Module System

Tau's module system is simple and file-based. Organize code naturally using folders and files without complex package managers, making it easy to structure and navigate your projects.
